# Запуск и публикация

## Локальный просмотр

Исходный `index.html` нельзя открывать двойным кликом. Это входной файл React/Vite.

```powershell
npm.cmd install
npm.cmd run dev
```

Откройте адрес, который покажет терминал.

## Обычный хостинг

```powershell
npm.cmd run build
```

Загрузите на хостинг всё содержимое папки `dist`, включая папку `assets`.

## GitHub Pages

1. Загрузите весь проект в ветку `main` или `master`.
2. В репозитории откройте `Settings` -> `Pages`.
3. В поле `Source` выберите `GitHub Actions`.
4. Откройте вкладку `Actions` и дождитесь завершения `Deploy GitHub Pages`.

Не выбирайте публикацию исходной папки репозитория. GitHub Pages должен публиковать результат сборки из `dist`.
